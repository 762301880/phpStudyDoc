##  任务金币功能

```php
│   ├── 播放返金币系统
│   │   ├── 打开App播放 +1金币（每日5次）
│   │   ├── 每5分钟 +2金币
│   │   ├── 累计30分钟 +10金币 + 分享弹窗
│   │   ├── 分享作品 +5金币（每日3次）
│   │   └── 裂变：双方各+20金币
```

例如以上功能建表

**用户金币信息表**

```sql
CREATE TABLE `fa_user_gold` (
  `id` bigint unsigned NOT NULL AUTO_INCREMENT COMMENT '主键',
  `user_id` int NOT NULL COMMENT '用户ID',
  `gold` int NOT NULL DEFAULT '0' COMMENT '当前可用金币(后期需要加已使用金币记录表)',
  `total_gold` int NOT NULL DEFAULT '0' COMMENT '累计总获得金币',
  `create_time` datetime NOT NULL DEFAULT '0000-00-00 00:00:00',
  `update_time` datetime NOT NULL DEFAULT '0000-00-00 00:00:00',
  PRIMARY KEY (`id`),
  UNIQUE KEY `idx_uid` (`user_id`)
) ENGINE=InnoDB AUTO_INCREMENT=4 DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_0900_ai_ci COMMENT='用户金币信息表';
```

**金币流水表**

```sql
CREATE TABLE `fa_user_gold_log` (
  `id` bigint unsigned NOT NULL AUTO_INCREMENT COMMENT '自增主键',
  `user_id` int NOT NULL COMMENT '用户ID',
  `gold_num` int NOT NULL COMMENT '金币变动数量，正数=收入，负数=支出',
  `log_type` tinyint NOT NULL DEFAULT '1' COMMENT '收支类型：1收入(默认) 2支出',
  `rel_id` bigint DEFAULT '0' COMMENT '关联业务ID：分享ID/裂变ID/播放任务ID',
  `remark` varchar(200) DEFAULT '' COMMENT '前端展示文案，如「打开APP播放奖励+1金币」',
  `create_time` datetime NOT NULL DEFAULT '0000-00-00 00:00:00' COMMENT '时间戳',
  `update_time` datetime DEFAULT NULL,
  PRIMARY KEY (`id`),
  KEY `idx_uid_ctime` (`user_id`,`create_time`)
) ENGINE=InnoDB AUTO_INCREMENT=27 DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_0900_ai_ci COMMENT='用户金币变动流水表';
```

**每日任务表**

```sql
CREATE TABLE `fa_user_play_task` (
  `id` bigint unsigned NOT NULL AUTO_INCREMENT COMMENT '主键ID',
  `user_id` int NOT NULL COMMENT '用户ID',
  `task_date` date NOT NULL COMMENT '任务日期 yyyy-MM-dd',
  `open_app_count` tinyint NOT NULL DEFAULT '0' COMMENT '今日打开App播放奖励领取次数（上限5）',
  `total_play_min` int NOT NULL DEFAULT '0' COMMENT '今日累计播放总分钟数(展示X秒)',
  `five_min_reward_times` int NOT NULL DEFAULT '0' COMMENT '5分钟奖励领取总次数',
  `last_five_min_time` int NOT NULL DEFAULT '0' COMMENT '上一次领取5分钟奖励时间戳，防重复刷',
  `thirty_min_reward_status` tinyint NOT NULL DEFAULT '0' COMMENT '30分钟累计奖励：0未领 1已领',
  `share_count` tinyint NOT NULL DEFAULT '0' COMMENT '今日普通分享次数（上限3）',
  `create_time` datetime NOT NULL DEFAULT '0000-00-00 00:00:00',
  `update_time` datetime NOT NULL DEFAULT '0000-00-00 00:00:00',
  PRIMARY KEY (`id`),
  UNIQUE KEY `idx_user_date` (`user_id`,`task_date`)
) ENGINE=InnoDB AUTO_INCREMENT=21 DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_0900_ai_ci COMMENT='用户每日播放任务进度表\r\n\r\n每日任务限制表，控制每日领取次数、时长、防重复领奖';
```
