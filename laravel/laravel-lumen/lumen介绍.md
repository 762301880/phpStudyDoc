# 二: 安装&运行

## 2.1安装

* 我们采用`composer`安装

```php
composer create-project --prefer-dist laravel/lumen blog  #blog==项目名称
```

* 更多安装方法请参考[Lumen文档](https://learnku.com/docs/lumen/6.x/installation/6101#e655a42)

## 2.2运行

* 推荐采用集成环境这里我们采用[phpstudy](https://www.xp.cn/download.html)
