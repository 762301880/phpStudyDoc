## 收费标准

**拍云 USS（对象存储）** 的收费，目前官方标准价格大致如下：

| 存储类型    | 费用                                 |
| ----------- | ------------------------------------ |
| 标准存储    | 0.129 元/GB/月（约 0.0043 元/GB/天） |
| 低频存储    | 0.081 元/GB/月（约 0.0027 元/GB/天） |
| 上传流量    | 免费                                 |
| 请求次数    | 免费                                 |
| CDN回源流量 | 免费                                 |

### 标准存储（最常用）

有一个比较特殊的规则：

- 当天存储容量 ≤ 当天 CDN 流量时，存储免费；
- 超出部分才按 0.129 元/GB/月计费。 

例如：

- 存储 100GB
- 当天 CDN 流量 120GB

则存储费用为 0 元。

如果：

- 存储 100GB
- 当天 CDN 流量 20GB

则按超出的 80GB 计费。 

### 低频存储

适合备份、归档等不经常访问的数据：

| 项目         | 价格           |
| ------------ | -------------- |
| 存储         | 0.081 元/GB/月 |
| 数据取回     | 0.0325 元/GB   |
| 公网流出流量 | 0.5 元/GB      |

注意：

- 最低存储周期 30 天；
- 小于 64KB 的文件按 64KB 计算。 

### CDN 流量费用（如果绑定 CDN）

国内节点：

| 项目      | 价格         |
| --------- | ------------ |
| 流出流量  | 0.29 元/GB   |
| 动态请求  | 0.02 元/千次 |
| HTTPS请求 | 0.05 元/万次 |

### 实际成本举例

如果用于网站图片存储：

- 图片总量：50GB
- 每月 CDN 流量：300GB

由于流量远大于存储量，标准存储部分通常免费，只需支付 CDN 流量费用：

300 × 0.29 ≈ **87 元/月**。
