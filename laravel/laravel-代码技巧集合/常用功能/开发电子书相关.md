万章小说电子书分片保存落地方案（性能优化 + 避坑指南）

## 一、结论先行

**一万章小说采用「一章一个 TXT 分片」是行业标准做法，完全可行。**

但如果不做分层优化，会出现：目录臃肿、文件系统卡顿、数据库批量插入锁表、大文本内存溢出、接口 30s 超时等线上问题。

以下为可直接上线的完整落地方案。

## 二、万章分片 3 大核心问题 + 根治方案

### 1. 本地生成上万小文件，单目录拥堵、读写慢、Windows 卡顿

**问题**：单目录存放 10000+ 小文件，文件系统检索、遍历、IO 急剧变慢。

**解决方案：分目录归档（本地 / OSS 通用）**

按「每 1000 章一个文件夹」做分层归档，禁止全部堆在同一目录。

**目录结构规范**

```Plain
book/1001/
  0001/  # 1-999章
  0002/  # 1000-1999章
  0003/  # 2000-2999章
```

代码根据章节序号自动创建子目录，彻底解决单目录文件过多卡死问题。

### 2. 一次性 10000 条 SQL 批量入库，锁表、卡顿、超时

**问题**：单条 `insertAll` 一次性写入万条数据，事务过大、锁表时间长、接口超时。

**解决方案：分批批量插入，每批 1000 条**

拆分大批次为小批次，数据库 IO 平滑，不锁表、稳吞吐。

**PHP 标准代码**

```Plain
$insertBatch = [];
$batchSize = 1000;

foreach ($chapterList as $sort => $item) {
    $insertBatch[] = [...];

    // 满批次即入库
    if (count($insertBatch) >= $batchSize) {
        Db::name('book_chapter')->insertAll($insertBatch);
        $insertBatch = [];
    }
}

// 写入剩余数据
if (!empty($insertBatch)) {
    Db::name('book_chapter')->insertAll($insertBatch);
}
```

### 3. 整本大文本一次性读取，内存爆炸、30s 超时

**问题**：`file_get_contents` 一次性加载 16M+ 全书文本，内存暴涨、正则匹配极慢、直接超时。

**根治方案：流式逐行读取，不加载全书**

边读边识别章节、边分片，内存常驻仅几十 KB，彻底杜绝内存溢出与超时。

## 三、架构级避坑：分片任务必须丢消息队列

**严禁在上传 POST 接口内同步执行万章分片**（web 有执行时间、内存限制）。

**标准流程（FastAdmin / think-queue）**

1. 上传接口只接收文件、存临时文件、写入书籍主数据；
2. 将 `临时路径 + book_id` 推送队列，接口立刻返回成功；
3. CLI 消费队列（无执行超时、内存限制宽松），后台慢慢分片；
4. 前端轮询「书籍分片状态接口」，完成后再展示目录。

## 四、线上 OSS 万级小文件生产优化

针对一万个章节小文件的 OSS 存储专项优化：

1. **开启生命周期冷热分层**：99% 章节为冷门资源，自动归档低频存储，大幅降本；
2. **开启自动 gzip 压缩**：单章 txt 压缩后仅几 KB，整体流量极小；
3. **签名 URL 防盗链**：不暴露永久资源地址，防止爬虫批量爬取全站章节。

## 五、UniApp 前端万章加载体验优化

1. **章节列表分页查询**：不一次性返回 10000 条，每页 200 条滚动加载；
2. **按需预加载**：仅预缓存当前章节 + 后 2 章，不批量下载；
3. **本地 LRU 缓存**：手机只缓存最近 20 章，自动淘汰旧数据，不占用存储。

## 六、极端兜底：超大章节二次分片

针对单章几十万字的超长章节：

在生成 `chapter_xxxx.txt` 之前，按 **3000 字**二次切分小段，避免前端单文件过大、渲染卡顿、内存飙升。

## 七、企业级完整业务流程

1. 用户上传完整 TXT 电子书，接口接收并保存临时文件；
2. 创建 book 主表记录，推送分片任务至 Redis 队列，前端立即返回成功；
3. CLI 队列消费者**流式读取**全文，逐行识别章节；
4. 单章生成独立 TXT，按千章分目录上传 OSS；
5. 每 1000 条分批入库 `book_chapter`；
6. 全部处理完毕，更新书籍状态为「分片完成」；
7. UniApp 前端分页拉取章节列表，按需加载阅读。

## 步骤流程图

```php

      │
      ▼

      │
      ▼

      │
      ▼

      │
      ├── 小说信息

      └── 章节内容
      │

      │
      ▼

```

## 示例php代码逻辑

```php
<?php

namespace app\common\service;

class TxtSplitService
{
    // 正则扩容到50字符，兼容超长章节标题
    private $chapterReg = '/(第[0-9一二三四五六七八九十百千万]+章[^\n]{0,50})/u';

    /**
     * 主入口：拆分完整TXT
     * @param string $filePath 上传后的完整txt本地路径
     */
    public function split(string $filePath)
    {
        // 取消脚本超时限制
        set_time_limit(0);
// 取消内存限制（大文本读取占用内存高）
        ini_set('memory_limit', '300M');
        // 文件不存在校验
        if (!file_exists($filePath)) {
            throw new \Exception('源TXT文件不存在');
        }

        // 1. 读取文件并统一转UTF-8编码
        $content = $this->readFileToUtf8($filePath);

        if (empty($content)) {
            throw new \Exception('文件内容为空或读取失败');
        }

        // 2. 文本清洗：多余空行、特殊符号
        $content = $this->cleanText($content);

        // 3. 正则分割章节
        $chapterList = $this->splitByChapter($content);
        if (empty($chapterList)) {
            throw new \Exception('未识别到任何章节，可改用固定字数分片');
        }

        return $chapterList;
    }

    /**
     * 自动识别编码，转为UTF-8
     */
    protected function readFileToUtf8(string $path): string
    {
        $raw = file_get_contents($path);
        $encode = mb_detect_encoding($raw, ['UTF-8', 'GBK', 'GB2312', 'BIG5']); //检测字符串编码
        if ($encode !== 'UTF-8') {
            $raw = mb_convert_encoding($raw, 'UTF-8', $encode);
        }
        return $raw;
    }

    /**
     * 清洗文本：去除大量空行、多余空格
     */
    protected function cleanText(string $text): string
    {
        // 多个换行替换为单个换行
        $text = preg_replace("/\n\s+/", "\n", $text);
        // 去除首尾空白
        $text = trim($text);
        return $text;
    }

    /**
     * PHP7 兼容分割章节（修复原preg_split参数错误）
     * 返回 [ ['title'=>'章节名','content'=>'正文'] ]
     */
    /**
     * 只提取章节内容，自动丢弃文件开头版权、前言、广告
     * @param string $text 完整小说文本
     * @return array
     */
    protected function splitByChapter(string $text): array
    {
        $res = [];
        $textByte = $text;
        // 匹配全部章节标题，带字节偏移
        preg_match_all($this->chapterReg, $textByte, $matchAll, PREG_OFFSET_CAPTURE);

        // 未识别到任何章节直接返回空
        if (empty($matchAll[1])) {
            return [];
        }

        // 整理所有章节点位：标题、标题起始字节、标题结束字节
        $chapterList = [];
        foreach ($matchAll[1] as $matchItem) {
            $titleStr = trim($matchItem[0]);
            $titleStartByte = $matchItem[1];
            $titleByteLength = strlen($matchItem[0]);
            $titleEndByte = $titleStartByte + $titleByteLength;

            $chapterList[] = [
                'title' => $titleStr,
                'title_start' => $titleStartByte,
                'title_end' => $titleEndByte
            ];
        }

        $total = count($chapterList);
        // 循环拆分每一章正文
        for ($i = 0; $i < $total; $i++) {
            $current = $chapterList[$i];
            // 正文起始：当前章节标题结束的位置
            $contentStart = $current['title_end'];
            // 正文截止：下一章标题开头，最后一章到全文末尾
            if ($i < $total - 1) {
                $contentStop = $chapterList[$i + 1]['title_start'];
            } else {
                $contentStop = strlen($textByte);
            }

            // 按字节截取章节正文
            $chapterContent = substr($textByte, $contentStart, $contentStop - $contentStart);
            $chapterContent = trim($chapterContent);

            // 仅正文不为空才加入结果集
            if (!empty($chapterContent)) {
                $res[] = [
                    'title' => $current['title'],
                    'content' => $chapterContent
                ];
            }
        }

        return $res;
    }

    /**
     * 备用：无章节小说固定字数分片（3000字/片）
     */
    public function splitByWord(string $text, int $limit = 3000): array
    {
        $res = [];
        $total = mb_strlen($text);
        $index = 0;
        $part = 1;
        while ($index < $total) {
            $slice = mb_substr($text, $index, $limit);
            $res[] = [
                'title' => '第' . $part . '部分',
                'content' => $slice
            ];
            $index += $limit;
            $part++;
        }
        return $res;
    }
}
```

### 代码优化

```php
<?php

namespace app\common\service;

class TxtSplitService
{
    // 匹配卷标：第二卷/第2卷 兼容空格冒号横线
    private string $volumeReg = '/^[\s\-]*第([0-9一二三四五六七八九十百千万]+)卷\s*[:：]\s*(.+?)\s*$/um';
    // 匹配章节，兼容前后横线、大量空格、冒号、换行
    private string $chapterReg = '/[\-—\s]*第([0-9一二三四五六七八九十百千万]+)章\s*[:：]?\s*(.{0,50}?)\s*$/um';

    /**
     * 中文数字转阿拉伯数字（一卷、一百二十三章 → 1、123）
     */
    protected function cnToNum(string $cn): int
    {
        $cnMap = [
            '零' => 0, '一' => 1, '二' => 2, '三' => 3, '四' => 4, '五' => 5,
            '六' => 6, '七' => 7, '八' => 8, '九' => 9, '十' => 10,
            '百' => 100, '千' => 1000, '万' => 10000
        ];
        // 纯数字直接返回
        if (ctype_digit($cn)) {
            return (int)$cn;
        }
        $total = 0;
        $current = 0;
        $high = 0;
        for ($i = 0; $i < mb_strlen($cn); $i++) {
            $char = mb_substr($cn, $i, 1);
            $val = $cnMap[$char];
            if ($val >= 10) {
                if ($current === 0) $current = 1;
                $high = $current * $val;
                $total += $high;
                $current = 0;
            } else {
                $current = $val;
            }
        }
        $total += $current;
        return $total;
    }

    /**
     * 主入口：按章节流式拆分txt，低内存逐章节生成器输出
     * @param string $filePath 文件绝对路径
     * @return \Generator
     * yield数组携带：卷名、原文章节标题、统一连续章节号、正文内容
     */
    public function split(string $filePath): \Generator
    {
        set_time_limit(0);
        ini_set('memory_limit', '300M');

        if (!file_exists($filePath)) {
            throw new \Exception('源TXT文件不存在');
        }
        $handle = fopen($filePath, 'r');
        if (!$handle) {
            throw new \Exception('文件打开失败，请检查文件权限');
        }

        $chunkSize = 4096 * 64;
        $buffer = '';
        // 全局连续章节计数（入库使用，强制从1开始递增，不受分卷影响）
        $globalChapterNo = 0;
        // 当前卷名称
        $currentVolume = '无';

        while (!feof($handle)) {
            $chunk = fread($handle, $chunkSize);
            $chunk = $this->rawToUtf8($chunk);
            $buffer .= $chunk;
            unset($chunk);

            // 循环处理缓冲区所有完整行（先识别卷，再识别章节）
            while (true) {
                // 匹配换行，取出第一行处理
                $lineBreakPos = strpos($buffer, "\n");
                if ($lineBreakPos === false) {
                    break;
                }
                $oneLine = substr($buffer, 0, $lineBreakPos + 1);
                $remainBuffer = substr($buffer, $lineBreakPos + 1);

                // 1. 判断是否是卷标题
                if (preg_match($this->volumeReg, $oneLine, $volMatch)) {
                    $volNumCn = $volMatch[1];
                    $volName = trim($volMatch[2]);
                    $currentVolume = "第" . $this->cnToNum($volNumCn) . "卷 {$volName}";
                    $buffer = $remainBuffer;
                    continue;
                }

                // 2. 判断是否是章节标题
                if (preg_match($this->chapterReg, $oneLine, $chapMatch)) {
                    // 取出章节数字、章节原名
                    $chapNumCn = $chapMatch[1];
                    $chapRawTitle = trim($chapMatch[2]);
                    $buffer = $remainBuffer;

                    // 收集本章正文，直到下一章/下一卷
                    $chapterContent = '';
                    while (true) {
                        $nextLineBreak = strpos($buffer, "\n");
                        if ($nextLineBreak === false) break;
                        $checkLine = substr($buffer, 0, $nextLineBreak + 1);
                        // 遇到下一卷/下一章，停止收集正文
                        if (preg_match($this->volumeReg, $checkLine) || preg_match($this->chapterReg, $checkLine)) {
                            break;
                        }
                        $chapterContent .= $checkLine;
                        $buffer = substr($buffer, $nextLineBreak + 1);
                    }

                    // 全局章节自增（入库主键序号，统一1、2、3...）
                    $globalChapterNo++;
                    $cleanContent = $this->cleanText($chapterContent);
                    if (!empty($cleanContent)) {
                        yield [
                            'volume'       => $currentVolume,       // 所属卷，入库存卷字段
                            'raw_title'    => "第{$chapNumCn}章 {$chapRawTitle}", // 原文标题展示
                            'chapter_num'  => $globalChapterNo,     // 统一连续章节号（入库排序用）
                            'content'      => $cleanContent
                        ];
                    }
                    unset($chapterContent, $cleanContent);
                    continue;
                }

                // 普通正文行，放回缓冲区末尾
                $buffer = $remainBuffer;
            }
        }

        fclose($handle);

        // 文件末尾剩余文本作为尾声
        $lastText = trim($buffer);
        if (!empty($lastText)) {
            $globalChapterNo++;
            yield [
                'volume'       => $currentVolume,
                'raw_title'    => '尾声',
                'chapter_num'  => $globalChapterNo,
                'content'      => $this->cleanText($lastText)
            ];
        }
        unset($buffer);
    }

    /**
     * 无章节小说：按固定字数切割，生成器输出
     * @param string $text 完整utf8文本
     * @param int $limit 单段字数
     * @return \Generator
     */
    public function splitByWord(string $text, int $limit = 3000): \Generator
    {
        $text = $this->cleanText($text);
        $totalLen = mb_strlen($text);
        $offset = 0;
        $partNo = 1;

        while ($offset < $totalLen) {
            $slice = mb_substr($text, $offset, $limit);
            yield [
                'volume'       => '无',
                'raw_title'    => "第{$partNo}部分",
                'chapter_num'  => $partNo,
                'content'      => $slice
            ];
            $offset += $limit;
            $partNo++;
            unset($slice);
        }
    }

    /**
     * 自动编码识别，统一转UTF-8
     */
    /**
     * 自动编码识别，统一转UTF-8（修复分段截断残缺多字节报错）
     */
    protected function rawToUtf8(string $raw): string
    {
        if ($raw === '') return '';

        $detectList = ['UTF-8', 'GBK', 'GB2312', 'BIG5'];
        // 屏蔽iconv警告
        $oldErrorLevel = error_reporting();
        error_reporting($oldErrorLevel & ~E_NOTICE & ~E_WARNING);

        $encode = mb_detect_encoding($raw, $detectList, true);
        if (empty($encode)) {
            $encode = 'GBK';
        }

        if ($encode !== 'UTF-8') {
            // 第四个参数 IGNORE：丢弃无法转换的残缺字节
            $raw = mb_convert_encoding($raw, 'UTF-8', $encode, 'IGNORE');
        }

        error_reporting($oldErrorLevel);
        return $raw;
    }

    /**
     * 文本清洗：合并空行、去除首尾空白
     */
    protected function cleanText(string $text): string
    {
        // 多个换行/空白统一替换为单个换行
        $text = preg_replace('/\n\s+/', "\n", $text);
        $text = trim($text);
        return $text;
    }

    /**
     * 辅助方法：读取整个文件为UTF8文本（供splitByWord使用）
     * 注意：超大文件不要调用此方法，会一次性加载全文
     */
    public function getFullUtf8Content(string $filePath): string
    {
        $handle = fopen($filePath, 'r');
        $all = '';
        $chunkSize = 4096 * 64;
        while (!feof($handle)) {
            $all .= fread($handle, $chunkSize);
        }
        fclose($handle);
        return $this->rawToUtf8($all);
    }
}
```
