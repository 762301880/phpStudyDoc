##  电子书如何保存

## 分片保存

一万多章小说，按章节分片完全可行，讲清企业落地方案、性能优化、避坑点

一、先说结论

一万章**一章一个 txt 分片是行业标准做法**，完全没问题，只是要做多层优化，不然上传分片时超时、数据库批量插入卡顿、文件目录臃肿。

一、一万章会遇到的 3 个核心问题 + 解决方案

1. 本地临时目录一次性生成 10000 个小文件，读写慢、windows 卡顿

优化：分目录归档存储（本地 / OSS 都适用）

不要全部丢在 `book_1001/` 下，按千章分文件夹：

```php
book/1001/
  0001/chapter_0001.txt ~ chapter_0999.txt
  0002/chapter_1000.txt ~ chapter_1999.txt
  0003/chapter_2000.txt ~ chapter_2999.txt
```

代码生成文件名时判断 sort，自动创建子文件夹，避免单目录文件上万，文件系统检索卡死。

2. 批量插入 10000 条章节 SQL，单条 insertAll 压力大、数据库锁表

方案：分批批量插入，每次 1000 条

不要一次性把 1 万条塞进`insertAll`，循环每 1000 条执行一次入库：

```php
$insertBatch = [];
$batchSize = 1000;
foreach ($chapterList as $sort => $item) {
    // 组装单条数据
    $insertBatch[] = [...];
    // 满1000条就写入数据库，清空数组
    if(count($insertBatch) >= $batchSize){
        Db::name('book_chapter')->insertAll($insertBatch);
        $insertBatch = [];
    }
}
// 循环结束插入剩余不足1000条的数据
if(!empty($insertBatch)){
    Db::name('book_chapter')->insertAll($insertBatch);
}
```

好处：不会一次性锁表，数据库 IO 平缓，不会出现接口超时。

3. 16M + 一万章文本，一次性读取全文 + 正则匹配直接内存爆炸、30 秒超时

根治方案：**流式逐行读取 TXT，不一次性加载整本书**

你现在 `file_get_contents($path)` 一次性读取全部文本，一万章大文本直接占几百 M 内存，正则全局匹配巨慢。

改造读取逻辑：逐行读取，边读边识别章节，不用把整本小说放进内存，内存占用稳定几十 KB，不会超时。

**配套：分片任务丢消息队列（FastAdmin think-queue）**

一万章处理耗时极长，**绝对不能在 POST 上传接口同步执行分片**：

1. 上传接口只接收文件，把「临时文件路径 + book_id」丢队列，立刻返回成功给前端；
2. CLI 命令行执行队列消费（CLI 无 30 秒执行时间限制、内存限制更宽松）；
3. 前端增加「书籍分片状态查询接口」，轮询是否处理完成，完成后再展示章节目录。

## OSS 存储一万个小文件优化（线上生产必用）

1. 开启 OSS 生命周期冷热分层：一万章 99% 是冷门章节，自动归档低频存储，大幅降低存储成本；
2. OSS 文件 gzip 自动压缩：单章 txt 压缩后只有几 KB，一万章总流量极小；
3. 签名 URL 防盗链：不暴露永久文件地址，防止爬虫批量拉取一万章资源。

## 前端 uniapp 一万章加载体验优化

1. 目录接口分页查询：不要一次性返回 10000 条章节数据，分页每页 200 条滚动加载；
2. 预加载策略：只预缓存当前章节 + 后 2 章，不会一次性下载大量分片；
3. 本地 LRU 缓存：手机本地只缓存最近阅读的 20 章，自动清理旧分片，不占用手机存储。

## 极端兜底：单章超大（几十万字）二次分片

哪怕一万章，某一章十几万字，在生成 chapter_xxxx.txt 前，再按 3000 字二次切分小片段，前端阅读时内存不会卡顿。

## 一万章分片完整业务流程（企业标准）

1. 用户上传完整 TXT → 接口接收，保存临时文件；
2. 创建 book 书籍主记录，推送分片任务到 Redis 队列；接口返回成功；
3. 队列消费者（CLI）流式读取 txt，逐行识别章节；
4. 每一章生成独立分片文件，按千章分目录上传 OSS；
5. 每 1000 章批量插入 book_chapter 数据表；
6. 全部处理完成，修改 book 表状态为「已分片完成」；
7. uniapp 分页请求章节列表，按需加载对应分片 txt 阅读。

```php
<?php

namespace app\common\service;

//$file = $request->file('file');
//$originName = $file->getInfo()['name'] ?? "";//获取文件原始名称
//$savePath = './uploads/';
//$savefilePath = $savePath . $originName;

//txt分片实践 FastAdmin PHP7 适配版
class TxtSplitService
{
    // 正则扩容到50字符，兼容超长章节标题
    private $chapterReg = '/(第[0-9一二三四五六七八九十百千万]+章[^\n]{0,50})/u';

    /**
     * 主入口：拆分完整TXT
     * @param string $filePath 上传后的完整txt本地路径
     * @param int $bookId 书籍ID
     */
    public function split(string $filePath, int $bookId): bool
    {
        // 取消脚本超时限制
        set_time_limit(0);
// 取消内存限制（大文本读取占用内存高）
        ini_set('memory_limit', '256M');
        // 文件不存在校验
        if (!file_exists($filePath)) {
            throw new \Exception('源TXT文件不存在');
        }

        // 1. 读取文件并统一转UTF-8编码
        $content = $this->readFileToUtf8($filePath);
        if (empty($content)) {
            throw new \Exception('文件内容为空或读取失败');
        }

        // 2. 文本清洗：多余空行、特殊符号
        $content = $this->cleanText($content);

        // 3. 正则分割章节
        $chapterList = $this->splitByChapter($content);
        if (empty($chapterList)) {
            throw new \Exception('未识别到任何章节，可改用固定字数分片');
        }

        return $chapterList;
    }

    /**
     * 自动识别编码，转为UTF-8
     */
    protected function readFileToUtf8(string $path): string
    {
        $raw = file_get_contents($path);
        $encode = mb_detect_encoding($raw, ['UTF-8', 'GBK', 'GB2312', 'BIG5']); //检测字符串编码
        if ($encode !== 'UTF-8') {
            $raw = mb_convert_encoding($raw, 'UTF-8', $encode);
        }
        return $raw;
    }

    /**
     * 清洗文本：去除大量空行、多余空格
     */
    protected function cleanText(string $text): string
    {
        // 多个换行替换为单个换行
        $text = preg_replace("/\n\s+/", "\n", $text);
        // 去除首尾空白
        $text = trim($text);
        return $text;
    }

    /**
     * PHP7 兼容分割章节（修复原preg_split参数错误）
     * 返回 [ ['title'=>'章节名','content'=>'正文'] ]
     */
    /**
     * 只提取章节内容，自动丢弃文件开头版权、前言、广告
     * @param string $text 完整小说文本
     * @return array
     */
    protected function splitByChapter(string $text): array
    {
        $res = [];
        $textByte = $text;
        // 匹配全部章节标题，带字节偏移
        preg_match_all($this->chapterReg, $textByte, $matchAll, PREG_OFFSET_CAPTURE);

        // 未识别到任何章节直接返回空
        if (empty($matchAll[1])) {
            return [];
        }

        // 整理所有章节点位：标题、标题起始字节、标题结束字节
        $chapterList = [];
        foreach ($matchAll[1] as $matchItem) {
            $titleStr = trim($matchItem[0]);
            $titleStartByte = $matchItem[1];
            $titleByteLength = strlen($matchItem[0]);
            $titleEndByte = $titleStartByte + $titleByteLength;

            $chapterList[] = [
                'title' => $titleStr,
                'title_start' => $titleStartByte,
                'title_end' => $titleEndByte
            ];
        }

        $total = count($chapterList);
        // 循环拆分每一章正文
        for ($i = 0; $i < $total; $i++) {
            $current = $chapterList[$i];
            // 正文起始：当前章节标题结束的位置
            $contentStart = $current['title_end'];
            // 正文截止：下一章标题开头，最后一章到全文末尾
            if ($i < $total - 1) {
                $contentStop = $chapterList[$i + 1]['title_start'];
            } else {
                $contentStop = strlen($textByte);
            }

            // 按字节截取章节正文
            $chapterContent = substr($textByte, $contentStart, $contentStop - $contentStart);
            $chapterContent = trim($chapterContent);

            // 仅正文不为空才加入结果集
            if (!empty($chapterContent)) {
                $res[] = [
                    'title' => $current['title'],
                    'content' => $chapterContent
                ];
            }
        }

        return $res;
    }

    /**
     * 备用：无章节小说固定字数分片（3000字/片）
     */
    public function splitByWord(string $text, int $limit = 3000): array
    {
        $res = [];
        $total = mb_strlen($text);
        $index = 0;
        $part = 1;
        while ($index < $total) {
            $slice = mb_substr($text, $index, $limit);
            $res[] = [
                'title' => '第' . $part . '部分',
                'content' => $slice
            ];
            $index += $limit;
            $part++;
        }
        return $res;
    }
}
```
