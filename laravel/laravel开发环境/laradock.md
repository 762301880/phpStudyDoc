# 一、laradock简介

## 1.1介绍

> laradock是laravel 的docker开发环境，其中包含了打包好的常用的laravel开发环境，我们可以使用laradock很快的构建
>
> 自己的laravel开发环境，不需要依赖与系统环境

## 1.2 下载laradock

- 下载laradock

```php
git clone git@gitee.com:mirrors/Laradock.git
```

- 文档[地址](https://gitee.com/mirrors/Laradock#%E6%96%87%E6%A1%A3)
