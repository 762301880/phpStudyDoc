# [自定义多路由文件](https://www.kancloud.cn/manual/thinkphp5_1/353962)



```shell
├─route                 路由定义目录
│  ├─route.php          路由定义
│  ├─api.php            路由定义  # 名称可以自定义
│  └─...                更多路由定义
```

> 如下图所示我们自定义一个admin路由文件，访问的时候**you_server_url/test**即可访问路由 记住路由名称不要重复
