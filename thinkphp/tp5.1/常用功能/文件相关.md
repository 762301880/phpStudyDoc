#  获取文件后缀

```php
    /**
     * 获取文件后缀
     * @param $file
     */
    public static function getFileExt($file)
    {
       $fileInfo=$file->getInfo();
       $fileName=$fileInfo['name']??"";
       $fileNameArr= explode('.',$fileName);
       return end($fileNameArr);
    }
```

# 获取全部的文件名称

```php
    /**
     * 获取全部的文件名称
     * @param $file
     * @return mixed|string
     */
    public static function getOriginalName($file)
    {
        $fileInfo = $file->getInfo();
        $fileName = $fileInfo['name'] ?? "";
        return $fileName;
    }
```
