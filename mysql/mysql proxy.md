**简介**

> MySQL Proxy 是一个用于在 MySQL 服务器和客户端之间进行代理的工具，它可以用于实现负载均衡、故障转移、查询分析等功能。下面是 MySQL Proxy 的安装和使用步骤

## **下载**

> [点击下载地址](https://downloads.mysql.com/archives/proxy/) 下载对应的版本
