##   如何把index顶部自带的搜索添加编辑什么的拿掉然后自定义搜索

### 一、先区分两个要处理的区域

1. **左侧：刷新、添加、编辑、删除** 批量操作按钮（CRUD 自动生成在 index.html）
2. **右侧：原生搜索框、列显示、导出、刷新** 表格自带工具栏（bootstrapTable 控制）

### 第一步：移除左侧【添加 / 编辑 / 删除】按钮（修改 index.html 视图）

打开当前模块的 `view/xxx/index.html`

找到默认生成的工具栏代码：

```html
<div class="toolbar" id="toolbar">
    <button class="btn btn-success add"><i class="fa fa-plus"></i> 添加</button>
    <button class="btn btn-primary edit"><i class="fa fa-pencil"></i> 编辑</button>
    <button class="btn btn-danger del"><i class="fa fa-trash"></i> 删除</button>
</div>
```

### 两种处理方式

1. **直接删除**：删掉`add/edit/del`三个 button，只保留刷新（不需要刷新也一起删）

```html
<div class="toolbar" id="toolbar">
    <button class="btn btn-default refresh"><i class="fa fa-refresh"></i></button>
    <!-- 自定义搜索放这里 -->
</div>
```

1. **注释隐藏**：不想删就注释掉按钮代码即可

### 第二步：关闭表格原生右侧搜索、导出、列按钮（JS 表格初始化）

打开对应 js 文件（public/assets/js/backend/xxx.js），在`table.bootstrapTable({...})`配置里添加参数：

> [官方文档](https://doc.fastadmin.net/doc/190.html)

```js
table.bootstrapTable({
    url: $.fn.bootstrapTable.defaults.extend.index_url,
    pk: 'id',
    sortName: 'id',
    // ========== 关闭原生右侧工具栏组件 ==========
    search: false,        // 关闭原生单行搜索框
    showExport: false,    // 关闭导出按钮
    showColumns: false,   // 关闭列显示切换
    showToggle: false,    // 关闭表格视图切换
    commonSearch: false,  // 关闭顶部通用搜索表单
    // 其他原有配置...
    columns: [
        // 你的列配置
    ]
})
```

### 第三步：两种自定义搜索实现方案

### 方案 A：自定义顶部简单搜索框（放在左侧 toolbar 区域）

#### 1. 修改 index.html toolbar，加入自定义搜索表单

```html
<div class="toolbar" id="toolbar">
    <!-- 保留刷新 -->
    <button class="btn btn-default btn-refresh"><i class="fa fa-refresh"></i> 刷新</button>
    
    <!-- 自定义搜索区域 -->
    <div class="form-inline" style="display: inline-block;margin-left:10px;">
        <input type="text" id="search-name" class="form-control" placeholder="输入剧名搜索">
        <select id="search-type" class="form-control">
            <option value="">全部类型</option>
            <option value="小说">小说</option>
            <option value="短剧">短剧</option>
        </select>
        <button class="btn btn-primary" id="btn-search">搜索</button>
        <button class="btn btn-default" id="btn-reset">重置</button>
    </div>
</div>
<table id="table" class="table table-striped table-hover" data-operate-edit="{:$auth->check('xxx/edit')}" data-operate-del="{:$auth->check('xxx/del')}" width="100%"></table>
```

#### 2. JS 绑定搜索 / 重置事件

在 xxx.js 最底部添加：

```js
// 搜索按钮点击
$(document).on('click','#btn-search',function(){
    var name = $('#search-name').val();
    var type = $('#search-type').val();
    // 传递自定义参数到后台
    table.bootstrapTable('refresh',{
        query:{
            search_name: name,
            search_type: type
        }
    });
})
// 重置按钮
$(document).on('click','#btn-reset',function(){
    $('#search-name').val('');
    $('#search-type').val('');
    table.bootstrapTable('refresh',{query:{}});
})
// 回车触发搜索
$(document).on('keydown','#search-name',function(e){
    if(e.keyCode===13) $('#btn-search').click();
})
```

#### 3. 控制器接收搜索参数（后台过滤）

在对应控制器`index`方法里增加查询条件：

```js
public function index()
{
    if ($this->request->isAjax())
    {
        list($where, $sort, $order, $offset, $limit) = $this->buildparams();
        // 接收前端自定义搜索参数
        $search_name = $this->request->get('search_name','');
        $search_type = $this->request->get('search_type','');
        if($search_name) $where[] = ['user_search_name','like',"%$search_name%"];
        if($search_type) $where[] = ['search_type','=',$search_type];
        
        $list = $this->model
            ->where($where)
            ->order($sort,$order)
            ->limit($offset,$limit)
            ->select();
        $total = $this->model->where($where)->count();
        return json(['total'=>$total,'rows'=>$list]);
    }
    return $this->view->fetch();
}
```

### 方案 B：FastAdmin 官方通用搜索模板（多条件高级搜索）

适合多筛选条件，使用框架自带通用搜索渲染

1. JS 开启通用搜索并指定自定义模板

```js
table.bootstrapTable({
    url: $.fn.bootstrapTable.defaults.extend.index_url,
    pk: 'id',
    sortName: 'id',
    search: false,
    showExport: false,
    showColumns: false,
    showToggle: false,
    commonSearch: true, // 开启通用搜索
    searchFormVisible: true, // 默认展开搜索面板
    searchFormTemplate: 'customSearchTpl', // 自定义模板ID
    columns: [...]
})
```

2 .在 index.html 底部添加搜索模板

```html
<script id="customSearchTpl" type="text/html">
    <form class="form-commonsearch form-horizontal">
        <div class="row">
            <div class="col-xs-3">
                <label class="control-label">剧名</label>
                <input type="text" name="search_name" class="form-control">
            </div>
            <div class="col-xs-3">
                <label class="control-label">搜索类型</label>
                <select name="search_type" class="form-control">
                    <option value="">全部</option>
                    <option value="小说">小说</option>
                    <option value="短剧">短剧</option>
                </select>
            </div>
        </div>
    </form>
</script>
```

3. 控制器使用`buildparams`自动接收参数（无需手动获取）

```php
list($where, $sort, $order, $offset, $limit) = $this->buildparams();
// buildparams会自动读取通用搜索表单name参数，直接查询即可
$list = $this->model->where($where)->order($sort,$order)->limit($offset,$limit)->select();
```
