## 自定义验证类

### 验证器封装

```php
<?php
namespace app\admin\validate;
use think\Validate;

class User extends Validate
{
    protected $rule = [
        'name' => 'required|checkName', //绑定自定义方法checkName
        'mobile' => 'required|checkMobile',
    ];
    //错误提示
    protected $message = [
        'name.required' => '用户名不能为空',
    ];
    //场景：add新增/edit编辑分开校验
    protected $scene = [
        'add' => ['name','mobile'],
        'edit' => ['name','mobile'],
    ];

    /**
     * 自定义规则1：校验用户名不能重复（排除自身ID）
     * @param $value 当前字段值
     * @param $rule 规则参数
     * @param $data 整条提交表单数据
     * @return true|string
     */
    protected function checkName($value,$rule,$data)
    {
        $where = ['name'=>$value];
        if(!empty($data['id'])) $where['id'] = ['neq',$data['id']];
        $find = \app\admin\model\User::where($where)->find();
        return $find ? '用户名已存在' : true;
    }

    //自定义规则2：自定义手机号正则
    protected function checkMobile($value)
    {
        if(!preg_match('/^1[3-9]\d{9}$/',$value)){
            return '手机号格式错误';
        }
        return true;
    }
}
```

**调用**

> 步骤 3：控制器手动临时调用验证（自定义接口 / 非常规保存）

```php
public function test(){
    $post = $this->request->post();
    $validate = new UserValidate();
    if(!$validate->scene('add')->check($post)){
        $this->error($validate->getError());
    }
}
```
