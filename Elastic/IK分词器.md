# IK 分词器 到底是什么？

### ① 先搞懂 ES 原生的毛病

Elasticsearch 本身只认识**英文**：

比如英文：`iphone 15 pro`

它会自动拆开：`iphone`、`15`、`pro`，搜哪个都能找到。

但是！

**中文它完全不会！**

比如：`华为手机`

ES 原生会当成一整个字符串，不拆开！

你搜 `华为` 搜不到；搜 `手机` 也搜不到。

只能完整一模一样搜 `华为手机` 才找得到。

👉 **这就是 ES 中文废柴的原因。**

### ② IK 分词器 = 给 ES 装一个中文大脑

IK 分词器作用：

把中文自动拆开、分得很聪明：

```
华为新款智能手机
```

IK 会自动拆成：

```
华为`、`新款`、`智能`、`手机
```

然后用户随便搜：

搜 `华为`、搜`手机`、搜`智能`

**全部都能搜到！**

# 2. 一句话总结

- **没有 IK：中文搜索几乎不能用，很垃圾。**
- **装了 IK：商城中文搜索正常、好用、精准。**

# 3. 为什么你的商城必须装 IK？

你是商城：

商品标题：`超薄全面屏小米14手机`

没有 IK：

用户搜 `小米` → 搜不到

用户搜 `手机` → 搜不到

必须一模一样全打对才能搜到

有 IK 分词：

自动拆：超薄、全面屏、小米、14、手机

用户随便搜哪个词都能命中。

# 4. 超简单比喻（一秒懂）

- ES 原生：只会说英语，听不懂中文
- I 分词器：翻译 + 中文老师，教 ES 听懂中文

# 安装

## windows安装

## 第一步：找到你的 ES 安装文件夹

比如你桌面：

```
D:\elasticsearch-7.15.1
```

## 第二步：进入 plugins 文件夹

路径：

```
D:\elasticsearch-7.15.1\plugins
```

## 第三步：新建文件夹 名字必须叫 `ik`

plugins 里面创建：

```text
plugins\ik
```

## 第四步：下载对应版本 IK

https://release.infinilabs.com/analysis-ik/stable/

## 第五步：解压全部文件到 ik 文件夹

把 zip 里面**所有文件直接解压**放进：

```
D:\elasticsearch-7.15.1\plugins\ik
```

不要套二级文件夹！

## 第六步：重启你的 Elasticsearch

1. 关掉黑窗口 ES
2. 重新双击 `elasticsearch.bat` 启动

启动日志没有报错，就是装成功了 ✅
