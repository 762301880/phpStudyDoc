## `#[xxx]` 注解 这是什么

PHP **7.4 开始支持**，PHP8 正式普及。

写法就是：

```php
#[AutoController]
```

放在**类、方法、属性**上面，相当于**给代码打标记、贴标签**。

### 本质是什么？

不是注释！不是给人看的备注

是 **代码可以读取、程序能识别的特殊标记**。

就像给这个控制器贴了个牌子：

> 我这个类是**自动控制器**，框架你来自动帮我注册路由。

### php8注解解释

> 总览：https://www.php.net/manual/zh/language.attributes.overview.php
>
> 语法：https://www.php.net/manual/zh/language.attributes.syntax.php
>
> 预定义注解：https://www.php.net/manual/zh/language.attributes.predefined.php

**PHP8 原生语法，格式：**

```php
#[标签名(参数)]
```
