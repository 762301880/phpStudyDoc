## 触发远程构建

这是我们项目配置中的几个选项

```bash
General

源码管理

Triggers   # 在左边选择触发器选项

Environment

Build Steps

构建后操作
```

我们选择第一个 身份验证令牌是自定义的

![image-20260602140923423](https://gitee.com/yaolliuyang/blogImages/raw/master/blogImages/image-20260602140923423.png)

**然后会csrf导致无法构建成功**

例如上诉我们添加令牌后的令牌为

> 放到**gitee**项目的webhook还是无法执行

```php
# JENKINS_URL/job/vibe_tv/build?token=TOKEN_NAME

http://47.76.200.232:端口/job/vibe_tv/build?token=自定义token
```

需要安装插件**Build Authorization Token Root **

[官方文档](https://plugins.jenkins.io/build-token-root/)

> 一句话说：**这个插件就是让你 “不用登录 Jenkins，仅凭一个 secret token 就能远程触发构建”，专门解决 403 无权限、crumb 校验失败的问题。**Jenkins Plugins

改完后的webhookurl

```php
http://47.76.200.232:ip地址/buildByToken/build?job=vibe_tv&token=自定义token
```
