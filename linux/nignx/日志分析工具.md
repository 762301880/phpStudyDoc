# goaccess(推荐)

## 安装

```shell
# centos
sudo yum install goaccess
# ubuntu
sudo apt update && sudo apt install goaccess
# macos
brew install goaccess
```

## 基本使用

```shell
goaccess www3.puxiansheng.com_2025-08-08.log \
--log-format='%v %h %^[%d:%t %^] "%r" %s %b "%R" "%u" "%^" %^ %T %^' \
--date-format=%d/%b/%Y \
--time-format=%H:%M:%S \
-a > report.html
```
