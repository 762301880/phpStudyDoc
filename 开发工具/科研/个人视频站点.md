#  搭建个人视频站

## [LibreSpark/LibreTv](https://github.com/LibreSpark/LibreTv)

**docker-compose.yml**

```dockerfile
services:
  libretv:
    image: bestzwei/libretv:latest
    container_name: libretv
    ports:
      - "8899:8080" # 将内部 8080 端口映射到主机的 8899 端口
    environment:
      - PASSWORD=${PASSWORD:-111111} # 可将 111111 修改为你想要的密码，默认为 your_password(只要将-号后面的数字改为你需要的密码即可)
    restart: unless-stopped
```

## [synctv](https://github.com/synctv-org/synctv)

**docker-compose搭建**

```php
version: '3.3'
services:
  synctv:
    image: 'synctvorg/synctv:latest'
    container_name: synctv
    restart: unless-stopped
    ports:
      - '8081:8080/tcp'
      - '8081:8080/udp'
    volumes:
      - /opt/synctv:/root/.synctv
    environment:
      - PUID=0
      - PGID=0
      - UMASK=022
      - TZ=Asia/Shanghai
```

### Linux 一步到位（Cloudflare Tunnel

① 下载 cloudflared

直接用 curl：

```shell
curl -L https://github.com/cloudflare/cloudflared/releases/latest/download/cloudflared-linux-amd64 -o cloudflared
```

② 加执行权限

```shell
chmod +x cloudflared
```

③ （可选）放到全局

这样你以后随便在哪都能用：

```shell
sudo mv cloudflared /usr/local/bin/
```

④ 启动你的项目

比如：

```shell
npm run dev
```

假设你是：

```shell
http://localhost:5173
```

⑤ 开隧道（关键一步）

```shell
cloudflared tunnel --url http://localhost:5173

# 补充 自动常驻后台 -使用nohub命令

nohup cloudflared tunnel --url http://localhost:8081 > cf.log 2>&1 &

# 杀死进程
root@iZwz9f5r5auq14qsnsd44fZ:~# ps aux | grep cloudfla
root     4030619  0.2  2.2 1260800 37504 pts/1   Sl   18:23   0:00 cloudflared tunnel --url http://localhost:5173
root     4030724  0.0  0.1   6544  2304 pts/1    S+   18:24   0:00 grep --color=auto cloudfla
root@iZwz9f5r5auq14qsnsd44fZ:~# kill 4030619
```

👉 解释一下：

- `nohup`：关终端也不死
- `> cf.log`：输出写到日志
- `2>&1`：错误也写进去

👉 查看日志：

```
tail -f cf.log
```

🌐 成功后你会看到

类似：

```shell
https://xxxxx.trycloudflare.com
```

👉 用这个地址打开你的项目
 👉 **HTTPS 自动搞定**
 👉 摄像头 / 麦克风 API 就能用了 ✅
