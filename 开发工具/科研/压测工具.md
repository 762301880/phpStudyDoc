**汉化**

> 记事本打开**apache-jmeter-5.4.2\bin\jmeter.properties**

```shell
# 找到 language=en 下面添加一行 q
language=zh_cn
```



下载完成之后解压&启动

> 进入bin目录下启动命令

```shell
# 启动
sh jmeter
```
