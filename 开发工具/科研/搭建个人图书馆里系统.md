## 搭建

**docker搭建**

```dockerfile
docker run -d --name talebook -p 8080:80 -v /localdata:/data talebook/talebook
```

## 上传图书

> **<font color='red'>请将需要导入的书籍放入/data/books/imports/目录中。 支持的格式为 azw/azw3/epub/mobi/pdf/txt 。</font>**
